# 📝 Daily Task Manager

A simple and elegant to-do list web app made using **HTML, CSS, and JavaScript**.

## ✅ Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Auto-save using localStorage

2. Open `index.html` in any browser
3. Start managing your tasks!
